Save all powershell scripts available in DeploymentScripts folder into location Drive
Run Powershell ISE as Administrator and set path to local Drive (Scripts location)
Run ./InstallationScript
Wait until Script is Completed. Close Powershell ISE

Re-Open Powershell ISE as Administrator and set path to local Drive (Scripts location)
Run ./AADAppChanges -UserAADAppName "AADApplicationName" -AADWebAppID "AADWebApplicationID" -AADNativeAppID "AADNativeApplicationID" -ReplyUrl "CRM Base URL"
Note : Parameters are self-explanatory.
 - UserAADAppName : AADApplication Name provided in Dynamics 365 Azure set up tool
 - AADWebAppID : KeyVault application ID generated by Dynamics 365 Azure set up tool
 - AADNativeAppID : Native application ID generated by Dynamics 365 Azure set up tool
 - ReplyUrl : CRM Org Url
 
 Wait until script is completed and Run Below command 
./AzureResourceChanges -UserResourceGroupName "ResourceGroupName" -UserStorageAccountName "StorageAccountName" -UserKeyVaultName "KeyVaultName" -UserFunctionAppName "FunctionAppName" -UserPrincipalName "UserPrincipalName" -UserallowedOrigins "CRM Base URL"
Note : Parameters are self-explanatory.
 - UserResourceGroupName : Resource Group name provided in ARM Template
 - UserStorageAccountName : Storage Account Name provided in ARM Template
 - UserKeyVaultName : KeyVault Name provided in ARM Template
 - UserFunctionAppName : Function Application Name provided in ARM Template
 - UserPrincipalName : User id to be added in Keyvault
 - UserallowedOrigins : CRM Org Url
